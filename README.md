# glad_glyphs
A punchy little alliterative synonym generator for two-word queries. 

Ex: the query "kind engineer" results in the following results:

altruistic arranger  
attentive architect  
benignant builder  
charitable contriver  
courtly commander  
courteous captain  
dutiful developer  
deferential designer  
diplomatic director  
generous generator  
humane hatcher  
hospitable handler  
magnanimous maneuverer  
obliging organizer  
polite pioneer  
selfless schemer  
sympathetic spawner
